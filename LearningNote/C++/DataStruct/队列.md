# 队列
---
## 是什么
* 队列是一种特殊的线性表
* 仅能在线性表的两端进行操作
  * 队头(Front) : 取出数据元素的一端
  * 队尾(Rear) : 插入数据元素的一端

## 特性
先进先出(First In First Out) (FIFO)

## 操作
* 创建队列```Queue() ```
* 销毁队列```~Queue() ```
* 清空队列```clear() ```
* 进队列```add() ```
* 出队列```remove() ```
* 获取队头元素```front() ```
* 获取队列的长度   ```length() ```
