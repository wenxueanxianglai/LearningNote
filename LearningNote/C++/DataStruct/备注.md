## linkList 头结点隐患
```C++
template <typename T>
class LinkList : public List<T>
{
struct Node : public Object
{
  T value;
  Node* next{nullptr};
};

mutable Node m_header;
int m_length{0};
};
```
如果 T类型 初始化时候崩溃，则存在隐患

## 解决：
避免构造 T 类型(避免调用构造函数)，但内存布局一定要和原来一致
* 避免方法：
```C++
mutable struct {
  char reserved[sizeof(T)];
  Node* next;
} m_header;
```
