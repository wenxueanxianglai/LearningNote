# 线性表
---
## 1. 是什么?
特殊的数据对象
#### 表现：
* 零个，或 多个数据元素组成的集合
* 数据元素 在位置上 是有序排列
* 数据元素个数有限
* 数据元素类型相同

#### 抽象定义：
线性表 是： 具有相同类型的 n(>= 0) 个数据元素的有限序列
```
(a0, a1, a2, ... an-1)
ai 是表项 (数据元素)
n 是表长度
```

## 2. 有什么?
* a0 为线性表的第一个元素，只有一个后继
* an-1为线性表的最后一个元素，只有一个前驱
* 除 a0 和 an-1 外的 ai， 既有前驱，又有后继
* 直接支持 **逐项访问** 和 **顺序存取**


#### 操作
* 增(插入)
* 删（删除元素）
* 改（修改元素）
* 查（获取目标位置元素的值）
* 获取长度（获取表长度）
* 清空（清空线性表）
